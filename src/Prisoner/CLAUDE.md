# Prisoner Module

## Purpose
Prisoner and guest management.

## Files
**Patches:** PrisonerTabPatch.cs
**States:** PrisonerTabState.cs
**Helpers:** PrisonerTabHelper.cs

## Key Shortcuts
- **Arrow Keys** - Navigate prisoner settings
- **Enter** - Toggle options

## Architecture
Tab navigation for prisoner interaction, conversion, and release settings.

## Dependencies
**Requires:** ScreenReader/, Input/, Pawns/ (prisoner is a pawn)
